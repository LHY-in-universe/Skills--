# 蒙特卡洛积分技能 (Monte Carlo Integration Skill)

本技能通过随机数模拟估算一维定积分，内置五种经典蒙特卡洛方法，支持多函数选择与方法横向对比，所有结果以 JSON 格式输出。

## 数学背景

定积分 ∫[a,b] f(x)dx 的蒙特卡洛估计核心思想：

> 利用大数定律，用大量随机样本的函数值均值来近似积分期望值。误差以 O(1/√N) 收敛，与积分维度无关——这是蒙特卡洛方法在高维积分上胜过确定性方法的根本原因。

---

## 五种方法的数学原理

### 1. 基础随机采样（Crude Monte Carlo）

在区间 [a, b] 上均匀采样 N 个点 xᵢ，用函数均值乘以区间长度估计积分：

```
I ≈ (b-a) × (1/N) × Σ f(xᵢ)
```

最简单直接，方差为 Var(f) × (b-a)² / N，误差以 O(1/√N) 收敛。是其他所有方法的基准对照。

---

### 2. 打靶法（Hit-or-Miss）

在矩形区域 [a,b] × [0, y_max] 内均匀投点 (xᵢ, yᵢ)，统计落在函数曲线下方（yᵢ ≤ f(xᵢ)）的比例：

```
I ≈ (b-a) × y_max × (命中次数 / N)
```

几何直觉清晰，易于理解。但由于用矩形包围函数，浪费了大量"无效"采样点，方差通常高于基础方法，采样效率较低。

> 适用场景：函数形状未知、仅需要几何直觉验证时。

---

### 3. 分层采样（Stratified Sampling）

将积分区间 [a,b] 均匀划分为 K 层，在每层内独立均匀采样，再对各层结果加权求和：

```
I ≈ Σₖ [(b-a)/K × (1/nₖ) × Σᵢ∈层ₖ f(xᵢ)]
```

强制样本均匀分布，消除样本"聚集"现象。层数 K = √N 时理论方差最小。相比基础方法，方差可降低数倍至数十倍。

> 实现细节：本脚本取 K = max(10, √N)，每层采样数 N/K。

---

### 4. 重要性采样（Importance Sampling）

选择与 f(x) 形状相近的概率密度 g(x)，从 g 中采样 xᵢ，用比值修正：

```
I = ∫ [f(x)/g(x)] × g(x)dx ≈ (1/N) × Σ [f(xᵢ)/g(xᵢ)]，其中 xᵢ ~ g(x)
```

当 g(x) ∝ f(x) 时，f(x)/g(x) 接近常数，方差趋近于零。这是理论上方差缩减效果最强的方法。

> 实现细节：本脚本对所有函数使用截断指数分布（λ=2）作为 g(x)，通过逆变换法精确采样，无需拒绝采样。

---

### 5. 控制变量法（Control Variates）

选取已知精确积分的相似函数 h(x)（控制变量），利用 f 与 h 的相关性修正估计：

```
Î_cv = (b-a) × [f̄ - β × (h̄ - μ_h)]
其中 β = Cov(f, h) / Var(h)，μ_h = ∫h(x)dx/(b-a) 已知
```

通过引入已知量抵消随机误差，当 f 与 h 高度相关时方差大幅降低。β 由样本自动估计（最优系数）。

> 实现细节：本脚本以 h(x) = x 为控制变量，∫[0,1] x dx = 0.5 已知。

## 指令说明 (Instructions)

```
python3 scripts/monte_carlo.py --method <方法> --func <函数> --n <采样数> --seed <种子>
```

**参数说明：**

| 参数 | 可选值 | 默认值 | 说明 |
|------|--------|--------|------|
| `--method` | `crude` / `hit_or_miss` / `stratified` / `importance` / `control_variates` / `all` | `crude` | 积分方法；`all` 同时运行全部五种方法并按误差排序 |
| `--func` | `sin` / `x2` / `exp` | `sin` | 被积函数 |
| `--n` | 正整数 | `100000` | 随机采样点数，越大精度越高 |
| `--seed` | 整数 | `42` | 随机种子，固定后结果可复现 |

**输出字段说明：**

| 字段 | 含义 |
|------|------|
| `estimate` | 蒙特卡洛估计值 |
| `true_value` | 解析法精确值 |
| `abs_error` | 绝对误差 \|估计值 - 真实值\| |
| `rel_error` | 相对误差（小数形式） |
| `std_error` | 均值标准差（精度的统计量度） |

## 使用示例

> 任务：用基础随机采样法估算 ∫[0,1] sin(x) dx
> 1. 执行：
>    ```
>    python3 scripts/monte_carlo.py --method crude --func sin --n 100000 --seed 42
>    ```
> 2. 读取输出 JSON 中的 `estimate`（估计值）和 `abs_error`（绝对误差）。

---

> 任务：对比所有五种方法在 x² 积分上的精度，找出最优方法
> 1. 执行：
>    ```
>    python3 scripts/monte_carlo.py --method all --func x2 --n 50000 --seed 0
>    ```
> 2. 输出 JSON 中的 `comparison` 数组已按误差从小到大排序，`best_method` 字段直接给出最优方法名。

---

> 任务：用重要性采样法估算 ∫[0,1] eˣ dx，使用较少采样点验证效果
> 1. 执行：
>    ```
>    python3 scripts/monte_carlo.py --method importance --func exp --n 10000 --seed 123
>    ```
> 2. 对比 `abs_error` 与同等 n 下 crude 方法的误差，验证方差缩减效果。

## 方法选择指南

| 场景 | 推荐方法 |
|------|----------|
| 快速验证 / 基准测试 | `crude` |
| 需要几何直觉展示 | `hit_or_miss` |
| 通用精度提升（无需先验） | `stratified` |
| 已知函数形状，追求最低方差 | `importance` |
| 函数与线性/已知函数高度相关 | `control_variates` |
| 不确定选哪种 | `all`（自动找最优） |
